### **What is Node.js?**

**Node.js = A kitchen (environment) where your recipes (code) can run**

- **Free to download**
- **Works on any computer**
- **Lets JavaScript code run outside web browsers**
- **Required for our consciousness tools**

### **Step-by-Step: Install Node.js**

#### **Windows:**
1. Go to: https://nodejs.org/
2. Click the big green "Download" button
3. Run the installer (like installing any app)
4. Restart your computer

#### **Mac:**
1. Go to: https://nodejs.org/
2. Download the .pkg file
3. Double-click and follow instructions

#### **Linux:**
```bash
# Open terminal and type:
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs
```

#### **Test Installation:**
1. Open Command Prompt/Terminal
2. Type: `node --version`
3. You should see: `v18.17.0` (or higher)
4. Type: `npm --version`
5. You should see a version number

## üÜò **Chapter 9: Getting Help**

### **Common Problems:**

#### **"npm install fails"**
```
Solution:
1. Check internet connection
2. Try: npm install --force
3. Delete node_modules folder and try again
```

#### **"Server won't start"**
```
Solution:
1. Check if port 3000 is free: netstat -an | find "3000"
2. Kill process using port 3000
3. Try different port: npm run serve -- --port 3001
```

#### **"AI can't connect"**
```
Solution:
1. Check server is running (you should see "Server running" message)
2. Verify MCP configuration path
3. Restart your AI assistant
```
